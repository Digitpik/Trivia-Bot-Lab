# Trivia-Bot-Lab
A functional JavaScript Greeting and Trivia Bot demonstrating variable declaration, concatenation, and reassignment logic.
JavaScript Trivia Bot Lab
Project Overview
This project is a functional JavaScript-based Greeting and Trivia Bot developed as part of the freeCodeCamp JavaScript certification. It demonstrates a solid understanding of fundamental programming concepts required for modern web development.

Key Technical Skills Demonstrated
Variable Declaration: Efficient use of const for static data and let for variables that require dynamic updates.

String Concatenation: Merging multiple strings and variables to create personalized, user-facing messages.

Variable Reassignment: Updating variable values (like codingFact and status) to manage state throughout the script.

Console Debugging: Utilizing console.log() to output formatted data and verify script logic.
